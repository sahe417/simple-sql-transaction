### Usage

```bash
usage: test.sh [ -t TABLE_NAME ] [ -s SID ] [ -i Interval ] [ -u DB USER ] [ -p DB PASSWORD ] [ -n NET IDENTIFIED ] [ -l ]
    -t table name default TEST_TABLE
    -s ORACLE_SID Default orcl
    -i Interval Time for Query Transaction
    -u Oracle Database User Default oracle
    -p Oracle Database User Password Default passwd
    -n Network Identified in tnsname.ora
    -l enbale logging mode
    -h Print option help.
```
